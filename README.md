# MDR STEPS - THE WEB APP
## SE APPEN

1) Klicka på den gröna knappen "Code" uppe till höger i ditt repo.
2) Välj fliken "Codespaces".
3) Klicka på den blå knappen "Create codespace on main".
4) När editorn har laddats i webbläsaren, öppna terminalen längst ner och skriv startkommandot npm run dev
5) En pop-up ruta dyker upp i det nedre högra hörnet. Klicka på **"Open in Browser"** för det att se appen live.
